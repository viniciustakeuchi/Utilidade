Guia de Extensões para o VS Code

Um atalho de teclado é um tipo de comando de interface do usuário que permite que uma única entrada de teclado seja feita para enviar um comando a um programa de computador ou aplicativo. Os atalhos também podem ser personalizáveis em um programa, permitindo que o usuário desse programa defina seus próprios atalhos para comandos que usa com frequência. Essencialmente, um atalho de teclado é qualquer comando executado com o pressionar de uma tecla do teclado, em vez de por meio de outra interação do usuário, como um mouse. Um atalho de teclado pode ser bastante simples e envolve o pressionamento de um único botão para inserir um comando além de criar uma letra ou número.

Extensões

Git Lens

Essa extensão é um combo recheado de recursos que te ajudam a compreender melhor o código. É possível visualizarmos todas alterações que foram feitas, quando e por quem, contribuindo para o trabalho em equipe. Além disso, traz diversas informações sobre o histórico do Git, ou seja, histórico de arquivos, comparações com versões anteriores e muito mais.

Auto Rename Tag

Quando estamos trabalhando com HTML e precisamos renomear uma Tag, basta renomear uma delas. O VS Code já faz isso para nós. Porém, quando estamos usando React e estamos escrevendo JSX, o VS Code ainda não renomeia ambas as tags. A fim de economizar tempo, basta instalar o Auto Rename Tag

Color Highlight

O Color Highlight é uma ótima extensão para front-ends. Ela identifica quando você escreve o código de uma cor e a exibe no editor. Assim fica fácil de saber de qual cor se trata cada código.

Live Server

Uma das extensões obrigatórias para quem trabalha com front-end através do Visual Studio Code. Ele cria um localhost da aplicação e toda alteração que é salva no projeto, é atualizada automaticamente, evitando dar refresh na página a cada modificação no código feita.

Beautify

É um dos plugins mais famosos dos desenvolvedores, ele é um embelezador de código, tornando ele identado para arquivosjson, html, css, Sass e javascript.

ESLint

Ele serve para criar regras e padrões para seu código Js (ponto e vírgula, indentação, etc.), alertando erros ou avisos caso o seu código esteja fora das regras definidas.

Git Tags

O plugin Git Tags encurta o processo de criação de tags com uma interface simples e intuitiva.
