Guia de Estudos

Nesta página falaremos sobre alguns sites, com o intuito de ajudar, complementar e adicionar um conhecimento extra de tecnologia e também, pode servir como guias para seus estudos na área da T.I.

W3schools

W3Schools é um site com o foco educacional voltado ao aprendizado de tecnologias web. Seu conteúdo inclui tutoriais e referências relacionadas ao

  • Html
  • Css
  • JavaScript
  • JSON
  • PHP
  • Python
  • AngularJS
  • SQL
  • Bootstrap
  • Node.js
  • jQuery
  • XQuery
  • AJAX
  • XML
  • Java

Rocketseat

Rocketseat: Rocketseat é uma plataforma de educação onde você encontra tudo o que precisa para evoluir na programação e alcançar seus objetivos. No rocketseat há aulas para dominar:

  • HTML
  • CSS
  • JavaScript
  • HTTP
  • Forms
  • Ferramentas
  • DOM
  • e muito mais

MDN

MDN Web Docs ou mais conhecido como Mozilla Developer Network, é o website oficial de Mozilla para desenvolvimento de padrões web e projetos.

Udemy

A Udemy é uma plataforma digital de educação que oferece milhares de cursos. Existem opções de cursos pagos e gratuitos. Na Udemy você encontrará uma variedade de cursos sobre desenvolvimento, onde você poderá aprender:

  • Desenvolvimento de sistemas
  • Desenvolvimento de games
  • Desenvolvimento de aplicativos e aplicações móveis
  • Criação de sites para E-commerce

UpSkill

O UpSkill é um site que tem como público alvo pessoas que não tem conhecimento ou habilidade no mundo da programação. A plataforma é uma espécie de campo de treinamento que leva o usuário do nível iniciante ao mais avançado com aulas em vídeo. Nele você aprenderá diversos tipos de habilidades da programação como: Desenvolvimento de plugins para WordPress

  • As melhores práticas do WordPress
  • Javascript
  • HTML5 e CSS3
  • PHP
  • MySQL
  • Node.js
  • Ruby on Rails

Hackr.io

Hackr.io é uma plataforma que oferece cursos com diversos tipos de linguagens de programação. Nele contém cursos pagos e gratuitos com a escolha de níveis ao iniciante até o avançado. você poderá aprender:

  • Python
  • JavaScript e Java
  • Desenvolvimento para Android
  • C e C++
  • PHP
  • Estrutura de Dados e Algoritmos
