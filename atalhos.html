O que são atalhos?

Um atalho de teclado é um tipo de comando de interface do usuário que permite que uma única entrada de teclado seja feita para enviar um comando a um programa de computador ou aplicativo. Os atalhos também podem ser personalizáveis em um programa, permitindo que o usuário desse programa defina seus próprios atalhos para comandos que usa com frequência. Essencialmente, um atalho de teclado é qualquer comando executado com o pressionar de uma tecla do teclado, em vez de por meio de outra interação do usuário, como um mouse. Um atalho de teclado pode ser bastante simples e envolve o pressionamento de um único botão para inserir um comando além de criar uma letra ou número.

Atalhos do "Windows" mais importantes:

  • Tecla Windows + L: bloqueia o computador;
  • Tecla Windows + I: abre as configurações do sistema;
  • Tecla Windows + M: minimiza todas as janelas do monitor;
  • Tecla Windows + R: executa um comando;
  • Tecla Windows + SHIFT + M: restaura todas as janelas minimizadas;
  • Tecla Windows + S: Abre o sistema de busca do Windows;
  • Tecla Windows + E: mostra o explorador de arquivos;
  • Tecla Windows + X: Abre um menu que facilita o acesso a ferramentas do sistema;
  • Tecla Windows + T: Alterna entre os itens abertos na barra de tarefas e no desktop
  • Tecla Windows + Seta pra Cima: Maximiza a janela

Atalhos do "Desktop" com mais utilidade:

  • Ctrl + C= Copiar;
  • Ctrl + V = Cole o conteúdo da área de transferência;
  • Ctrl + A = Selecionar tudo;
  • Ctrl + X = Recortar / Recorta os itens selecionados;
  • Ctrl + Z = Desfazer uma ação;
  • Ctrl + S = Salva um documento rapidamente;
  • Ctrl + T = Abrir uma nova guia no Navegador;
  • Ctrl + Shift + N = Criar uma pasta nova;
  • Alt + Tab = Alterna entre os programas abertos;
  • ALT + ESC = Transita entre as janelas abertas
  • Ctrl + A Selecione todo o conteúdo;
  • Ctrl + F4 Feche a janela ativa.
